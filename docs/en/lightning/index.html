Simplechain Lightning Client

How to run a Simplechain Lightning network node

Simplechain has now its own implementation of the lightning network, it uses the eclair client. Currently, this is only supported on testnet, let's check how to set it up and run our very own lightning network client!.

it is recommended to run Debian 10 (buster) as it has all the library versions we need

Install Simplechain Core

Let's go ahead and install Simplechain on our Debian 10, the easy way is to use the Simplechain Debian repository:

Enable the Simplechain repository

Installing dirmngr and apt-transport-https

These two packages are needed to enable the Simplechain repository on Debian, let's install them:

apt install -y apt-transport-https dirmngr

Adding repository to your APT sources.

sudo su - Sudo to root first

echo "deb https://repo.Simplechain.info/apt/debian/ buster main" >> /etc/apt/sources.list.d/Simplechain.list

Add Simplechain key

Before installing, we need to add the Simplechain apt key

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys BF5B197D

Refreshing APT sources and installing Simplechain

sudo apt update && sudo apt install Simplechain

Simplechain should be installed now, let's continue

Create Simplechain configuration file

We need to create the .Simplechain folder manually, also, we will create the configuration file inside of that folder, here's how:

mkdir ~/.Simplechain

nano ~./Simplechain/Simplechain.conf

This will create a text file using the nano editor, copy/paste the following inside that file:

server=1

rpcuser=foo

rpcpassword=bar

txindex=1

addresstype=bech32

zmqpubrawblock=tcp://127.0.0.1:29000

zmqpubrawtx=tcp://127.0.0.1:29000

Install Simplechain Eclair - Lightning client

Let's install some dependencies to prepare for the lightning client:

Install JDK and Maven

Simplechain Eclair is developed using the Scala language. To run Simplechain Eclair, you must first install the JDK, and OpenJDK 11 or above is recommended.

apt-get install default-jdk

apt install maven

Build Simplechain Eclair

Now that we have every dependency installed, it's time to build the Eclair lightning client for Simplechain. Again, inside our terminal we type:

  • git clone https://github.com/Simplechainproject/lightning-demo.git
  • cd lightning-demo
  • mvn install -DskipTests

You should see something like this, if there's no errors, we can proceed with creating the configuration file and launch Simplechain Eclair now.

Launch Simplechain Eclair lightning client

Start the client with the command java -jar ./eclair-node-gui/target/lightning-capsule.jar

You should see the eclair desktop client as on the screenshot below:

Create a lightning network channel

Right-click the lower left corner of the local Simplechain Eclair client and select “Copy URI” to get its node address (public key).

Click the “Channel” button in the upper left corner and select “Open Channel”.

In the pop-up page, fill in the URI of the channel partner, and the capacity of the channel, which is set to 10 Simplechain as an example, this will be the maximum capacity for this channel, this means, you can only transact up to 10 Simplechain.

Click “Connect” to create the channel. At this point, you need to wait for 6 blocks to confirm the transaction and the channel will be created successfully.

Lightning Network Payment

Once the channel is set up, you can use the Lightning Network for micropayments.

Click the “Channel” button in the upper left corner, select “Receive Payment”, fill in the receipt amount in the pop-up page, click the “Generate” button to get your own receipt address. Here we choose a very small amount of 1 Satoshi, which is equivalent to 0.00000001 Simplechain. Such a small transferred amount is unimaginable without using a lightning network.

Click the “Channel” button in the upper left corner, select “Send Payment”, fill in the other party’s lightning network receipt address and amount in the pop-up page, and click the “Send” button, then you can transfer tokens to the other party.

After the transfer is completed, the amount of the two parties’ changes instantaneously. Because this is a pure transaction off the chain, there is no need to wait for the block confirmation, and it is free of fees.

Close the channel for settlement

The channel can be closed when the channel is no longer needed.

Click the “Close” button to close the channel.

Once closed, the remaining Simplechain in the channel will be refunded to the origin Simplechain Core wallet.
