Launch Simplechain With Docker

How to run Simplechain node with docker

Quick start

This tutorial use Linux Ubuntu as example, OSX and windows platform are almost the same.

Suppose the reader has basic command line knowledge, and also have docker correctly installed and configured. If not, please learn about command line and Docker first.

For more details, please refer to github

Get docker image

You might take either way:

Pull a image from Public Docker hub

$ docker pull Simplechain/Simplechain

Or, build Simplechain image with provided Dockerfile

The Dockerfile can be downloaded on github: Dockerfile

$docker build --rm -t Simplechain/Simplechain .

For historical versions, please visit docker hub

Prepare data path and Simplechain.conf

In order to use user-defined config file, as well as save block chain data, -v option for docker is recommended.

First chose a path to save Simplechain block chain data:

sudo rm -rf /data/Simplechain-data
sudo mkdir -p /data/Simplechain-data
sudo chmod a+w /data/Simplechain-data

Create your config file, refer to the example [Simplechain.conf]!(https://github.com/Simplechainproject/Simplechain/blob/1a926b980f03e97322c7dd787835bec1730f35d2/contrib/debian/examples/Simplechain.conf). Note rpcuser and rpcpassword to required for later Simplechain-cli usage for docker, so it is better to set those two options. Then please create the file ${PWD}/Simplechain.conf with content:

rpcuser=Simplechain
rpcpassword=Simplechaintest

Launch Simplechaind

To launch Simplechain node:

## to launch Simplechaind
$ docker run -d --rm --name Simplechain_node \
             -v ${PWD}/Simplechain.conf:/root/.Simplechain/Simplechain.conf \
             -v /data/Simplechain-data/:/root/.Simplechain/ \
             Simplechain/Simplechain Simplechaind

## check docker processed
$ docker ps

## to stop Simplechaind
$ docker run -i --network container:Simplechain_node \
             -v ${PWD}/Simplechain.conf:/root/.Simplechain/Simplechain.conf \
             -v /data/Simplechain-data/:/root/.Simplechain/ \
             Simplechain/Simplechain Simplechain-cli stop

${PWD}/Simplechain.conf will be used, and blockchain data saved under /data/Simplechain-data/

Interact with Simplechaind using Simplechain-cli

Use following docker command to interact with your Simplechain node with Simplechain-cli:

$ docker run -i --network container:Simplechain_node \
             -v ${PWD}/Simplechain.conf:/root/.Simplechain/Simplechain.conf \
             -v /data/Simplechain-data/:/root/.Simplechain/ \
             Simplechain/Simplechain Simplechain-cli getinfo

For more Simplechain-cli commands, use:

$ docker run -i --network container:Simplechain_node \
             -v ${PWD}/Simplechain.conf:/root/.Simplechain/Simplechain.conf \
             -v /data/Simplechain-data/:/root/.Simplechain/ \
             Simplechain/Simplechain Simplechain-cli help
