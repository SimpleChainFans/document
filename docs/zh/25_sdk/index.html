SDK

SimpleChain区块链向外部暴露了接口，外部业务程序能够通过SimpleChain提供的SDK来调用这些接口。开发者只需要根据自身业务程序的要求，选择相应语言的SDK，用SDK提供的API进行编程，即可实现对区块链的操作。

对接应用

目前，SDK接口可实现的功能包括（但不限于）：

合约操作

合约编译、部署、查询 交易发送、上链通知、参数解析、回执解析

链管理

链状态查询、链参数设置 组员管理 权限设置

多种语言SDK

目前，SimpleChain 提供的SDK包括：

Java SDK（稳定、功能强大)

Python SDK（简单轻便）

Node-js SDK（简单轻便）
